PARA FAZEREM A VOSSA BASE DE DADOS!!!

Aviso: update-database pode dar mas se n der assim dá de certeza


PASSO A PASSO
1. Criar uma Base de dados
	SQL SERVER OBJECT EXPLORER -> SQL Server -> Databases  --(botão direito)-> Add New Database (ou equivalente a Criar nova base de dados)
		
	USEM O NOME: " DbLibSpace "

2. Update database e check das tabelas
	Fazer update-database e confirmar se existem as tabelas dso.Asp qualquer coisa

3. Correr o Query nessa DB
	Na nova base de dados que criaram, cliquem com botão direito e cliquem em "New Query" (em portugues é o primeiro que disser "Novo.." qualquer coisa )
	
	Nesse novo query, copiem o código do ficheiro "SQL_Query_LS.sql" e colem aí.
	Nao precisam de guardar o ficheiro. 
	Apenas corram o código.

	Boom, base de dados com as tabelas feitas

3. Certificar que "appsettings.json" está com "Database=DbLibSpace" mesmo nome da vossa base de dados

4. Dar update-database outra vez e testar
